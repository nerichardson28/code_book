Binary Code activity - Count the Dots Activity

We learned another offline coding activity where the students could learn how to count using binary numbers.
The question "Try figuring out the numbers 1,2,3,4,5 in order..can you work out a pattern or algorithim for
flipping over the cards to count up by one?" was asked as we were doing the activity.
Once again this tested the old brain capacity as I had never done much work with binary numbers or offline coding
activities such as this. But once I got the cards and had it explained again so that I could see it, I understood.
my partner and I were able to figure it out count to 5 in order using the cards.
